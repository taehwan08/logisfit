## 2. 설정 요약

### INSTALLED_APPS

| 구분 | 패키지 |
|------|--------|
| Django 기본 | admin, auth, contenttypes, sessions, messages, staticfiles, humanize |
| 서드파티 | `rest_framework`, `corsheaders`, `widget_tweaks`, `storages` |
| 로컬 앱 | `apps.accounts`, `apps.clients`, `apps.inspection`, `apps.inventory`, `apps.fulfillment` |
| 예정 (주석) | `apps.works`, `apps.storage`, `apps.invoices`, `apps.contracts` |

### DATABASE

- **개발**: SQLite (`db.sqlite3`)
- **프로덕션**: PostgreSQL (`DATABASE_URL` 환경변수, `dj-database-url`)
- 커넥션 풀: `conn_max_age=600`

### Celery

| 항목 | 값 |
|------|-----|
| Broker/Backend | Redis (`CELERY_BROKER_URL`, `CELERY_RESULT_BACKEND`) |
| Serializer | JSON |
| Timezone | Asia/Seoul |
| Task Time Limit | 30분 |

### 기타 주요 설정

- **AUTH_USER_MODEL**: `accounts.User` (이메일 기반 커스텀 유저)
- **SESSION_COOKIE_AGE**: 12시간 (슬라이딩 윈도우)
- **SingleSessionMiddleware**: 동시 로그인 방지
- **DRF**: Session + Basic 인증, 페이지네이션 20건
- **PWA 지원**: 서비스워커 + 매니페스트
